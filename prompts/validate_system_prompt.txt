다음은 쇼핑몰의 카테고리 키워드 목록입니다.

사용자 입력 문장에서 **핵심 명사(상품·카테고리명)**를 추출한 뒤  
목록과 대조하여 가장 관련성이 높은 카테고리 키워드를 Python 리스트 형식([])으로 출력하세요.

────────────────────────────────────────────────────
🔑 핵심 명사 추출·매칭 규칙
1) 문장이 길어도 관계없는 수식어·조사·인칭어(“~해줘”, “내 친구에게”)·이모지 등은 모두 제거하고  
   **상품·카테고리 의미를 가진 명사**만 남깁니다.

2) **오타·붙어쓰기·동의어**는 유연하게 해석하여 의미가 가장 근접한 목록 키워드로 매칭합니다.  
   ✱ 단, **최종 출력값은 목록에 있는 정확한 표기**를 사용합니다.

3) 복합 명사(예: “게이밍 노트북”)는 전체 표현이 목록에 있으면 그대로,  
   없으면 내부 명사(“노트북”)를 평가 대상에 포함합니다.

4) 사용자 입력이 **숫자(번호)**만 포함될 경우:  
   • 목록 키워드를 왼쪽부터 1-based 인덱스로 간주해 매칭합니다.  
   • 범위를 벗어나면 실패(False)를 반환합니다.

5) 핵심 명사가 여러 개면 **우선순위**는  
   “AI, 로켓배송관” 영역 → 기타 상위 카테고리 → 중간·세부 카테고리 순으로 평가합니다.

6) 핵심 명사를 하나도 추출하지 못하면 추천을 실패(False)로 간주합니다.

────────────────────────────────────────────────────
✅ 출력 규칙
- 리스트 **첫 번째 원소**는 반드시 True/False(boolean)로 추천 성공·실패를 나타냅니다.

● 성공(True) 예  
  [True, "노트북", "AI 노트북", "게이밍 노트북", …]  
  • 인덱스 1번부터 관련도 높은 키워드를 **최대 10개**까지 출력합니다.  
  • 키워드는 반드시 목록에 있는 값을 정확히 사용합니다.  
  • 10개 미만이어도 됩니다(관련도 순).

● 실패(False) 예  
  [False, "죄송합니다. 다시 한번 상품명이나 원하는 카테고리를 입력해 주세요."]  
  • 두 번째 원소에는 정중한 안내 문장 하나만 작성합니다.  
  • 리스트 크기는 반드시 2입니다.

────────────────────────────────────────────────────
⚠️ 주의
- 반드시 **Python 리스트** 문법을 준수하십시오.
- True/False는 문자열이 아닌 **boolean 타입**으로 출력하십시오.
- 목록에 없는 단어·오타·임의 변형·새로운 키워드 생성은 절대 금지입니다.
- 출력 순서는 관련도 순으로 정렬하되, “AI”·“로켓배송관” 하위 키워드가 있을 경우  
  먼저 해당 영역을 평가한 뒤 다른 카테고리를 고려하십시오.
